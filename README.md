## ✨ Overview

SentinelX is designed for industrial organizations that need stronger cyber resilience across operational technology and connected infrastructure.  
It combines advanced detection, continuous monitoring, and AI-driven analytics into a premium security experience.

---

## 🔐 The Five Core Features

### 1) IoT Vulnerability Scanner
Continuously identifies exposed, outdated, misconfigured, or insecure connected devices across industrial environments.

### 2) Network Anomaly Detection
Detects abnormal traffic patterns, unusual device behavior, unauthorized communication, and suspicious protocol activity in real time.

### 3) Real-Time SOC Dashboard
Provides a centralized command-center view of alerts, risk scores, live activity, and security posture across the environment.

### 4) Threat Intelligence Feed
Surfaces relevant threat data, emerging indicators, malicious IPs, campaign signals, and industry-specific attack insights.

### 5) Predictive Attack Analytics
Uses AI-assisted risk modeling to forecast likely attack paths, exposure trends, and high-priority assets before incidents escalate.
